## 🚀 Featured Projects
 
### 🏥 DENTIVISE — AI Clinical Diagnostic Platform
> *The flagship project. Built entirely independently.*
 
A production-ready full-stack AI system for automated dental radiograph analysis. Trained a custom **YOLOv11 object detection model in PyTorch** on 1,400+ annotated dental X-ray images across 9 diagnostic classes. Integrated **Google Gemini multimodal API** for real-time clinical report generation from raw detection outputs. Engineered a scalable **FastAPI** backend for AI inference orchestration and built the complete clinician-facing interface in **React + Tailwind CSS**.
 
`PyTorch` `YOLOv8` `Gemini API` `FastAPI` `React` `Tailwind CSS` `Python`
 
---
 
### 🌤️ Horizon Weather — Full-Stack Weather Intelligence Platform
> *Advanced real-time weather application with full CRUD architecture.*
 
A sophisticated full-stack weather platform built from scratch with **FastAPI (async Python)** backend, **React.js (Vite)** frontend, and persistent **SQLite** database with **Pydantic** schema validation. Features multiplexed location resolution supporting city names, zip codes, and GPS coordinates with intelligent fallback routing. Delivers real-time hourly data injection, 5-day predictive trend forecasting, dynamic visual theming with environmental safety alerts, live **Google Maps** integration, location media streaming, and a complete **CRUD** system with JSON/CSV data export.
 
`FastAPI` `React` `Vite` `SQLite` `Pydantic` `Tailwind CSS` `Google Maps API` `REST APIs`
